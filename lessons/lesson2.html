Basic Elements & Structural Tags

HTML tags act like container brackets around texts. They tell the browser whether a piece of text is a heading or a regular block paragraph.

1. Headings (<h1> to <h6>)

Headings establish hierarchy. <h1> represents the main heading on the page.

2. Paragraphs (<p>)

The <p> tag is used for blocks of standard text. Each paragraph starts on a new line.

HTML Snippet Box 
<!DOCTYPE html>
<html>
  <body>
    <h1>Welcome to My First Website!</h1>
    <p>This is a text paragraph element.</p>
  </body>
</html>
