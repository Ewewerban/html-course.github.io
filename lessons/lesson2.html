1. Headings (h1 - h6)

To create titles and subtitles on a page, we use tags from <h1> to <h6>. Remember the golden rule: the larger the number inside the tag, the smaller the text looks on screen!

<h1> Main Page Title (Biggest importance)

<h2> Section Subtitle (Medium)

<h3> Tiny subheader (Small)

2. Paragraphs and formatting

For common blocks of text we use the paragraph tag <p>. You can also styles separate words inside using:

<b> ➔ Bold text wrapper
<i> ➔ Italicized cursive text

3. Code Example
