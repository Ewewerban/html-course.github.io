Basic Elements & Structural Tags

HTML tags act like container brackets around texts. Headings (ranging from <h1> to <h6>) define structural topics, while paragraphs (<p>) hold your regular text blocks.

<h1>This is a Main Heading</h1>
<p>This is a text paragraph containing information.</p>

Task: Copy these core tags into your workspace editor component and watch how the preview box renders them dynamically!
