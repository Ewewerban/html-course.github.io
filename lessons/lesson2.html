1. Stwórz swój pierwszy plik 1. Create your first file

🌐
index.html

Główna strona internetowa musi nazywać się dokładnie index.html. Przeglądarki internetowe (takie jak Chrome czy Firefox) zawsze szukają pliku o tej nazwie, kiedy wchodzisz na jakikolwiek adres URL. The main page of any website must be named exactly index.html. Web browsers (like Chrome or Firefox) always look for a file with this name when you visit any URL.

2. Wklej podstawową strukturę 2. Paste the basic structure

Każdy plik HTML potrzebuje kilku podstawowych linijek kodu, aby działał poprawnie. Skopiuj poniższy kod za pomocą przycisku i wklej go do swojego pliku: Every HTML file needs a few basic lines of code to work properly. Copy the code below using the button and paste it into your file:

HTML (index.html) 
<!DOCTYPE html>
<html>
  <head>
    <title>HTML For Beginners</title>
  &------- Zmień tekst poniżej! ------->
  </head>
  <body>
    <h1>Witaj Świecie!</h1>
    <p>To moja pierwsza strona www.</p>
  </body>
</html>

3. Co oznaczają te znaczniki? 3. What do these tags mean?

Kod HTML buduje się za pomocą tzw. znaczników (tags) ukrytych w nawiasach ostrokątnych < >. Większość z nich musimy otworzyć i zamknąć (za pomocą ukośnika /). HTML code is built using so-called tags enclosed in angle brackets < >. Most of them need to be opened and then closed (using a forward slash /).

  • <!DOCTYPE html> - Informuje przeglądarkę, że używasz najnowszej wersji języka (HTML5). Informs the browser that you are using the newest version of the language (HTML5).
  • <html> ... </html> - To główny kontener. Wszystko, co piszesz, musi znaleźć się pomiędzy tymi znacznikami. The main container. Everything you write must go between these tags.
  • <head> ... </head> - "Głowa" strony. Tutaj lądują rzeczy niewidoczne bezpośrednio na ekranie, jak tytuł karty w przeglądarce (<title>). The "head" of the page. This is where things not directly visible on the screen go, like the browser tab title (<title>).
  • <body> ... </body> - "Ciało" strony. To tutaj piszesz wszystko, co użytkownik zobaczy na Twojej stronie (teksty, obrazki, linki). The "body" of the page. This is where you write everything the user will actually see on your website (texts, images, links).
