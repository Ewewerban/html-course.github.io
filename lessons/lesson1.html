1. Stworzyć plik index.html 1. Create index.html file

Aby stworzyć prostą stronę w HTML, musisz najpierw przygotować plik tekstowy i nazwać go dokładnie index.html. To będzie główna strona Twojej witryny. To create a simple website in HTML, you must first prepare a text file and name it exactly index.html. This will be the main page of your website.

HTML (Gotowy index.html) 
<!DOCTYPE html>
<html>
  <head>
    <title>Moja Pierwsza Strona</title>
  </head>
  <body>
    <h1>Witaj świecie!</h1>
    <p>To jest moja strona w stylu rysunkowym.</p>
  </body>
</html>
